## 1. Modules and Packages

### 1.1 Module System

One file = one module. The module name is the file name. No header files, no `mod.rs` convention.

```
src/
    main.fe          → module main
    parser.fe        → module parser
    net/
        http.fe      → module net.http
        tcp.fe       → module net.tcp
```

### 1.2 Importing

```ferrum
import std.collections.HashMap
import std.io.{read_line, write_line}
import std.net                        // import the module
import std.net as network             // alias
import std.prelude.*                  // glob import (discouraged outside preludes)
```

### 1.3 Visibility

```ferrum
pub fn public_function() { ... }     // visible outside the module
fn private_function() { ... }        // module-private (default)

pub type PublicType { ... }
type PrivateType { ... }

pub(crate) fn crate_visible() { ... }  // visible within the package
pub(super) fn parent_visible() { ... } // visible in parent module
```

### 1.4 Packages

A package is a directory with a `Ferrum.toml` manifest:

```toml
[package]
name    = "mypackage"
version = "0.1.0"
edition = "2025"

[registries]
default = "https://packages.ferrum-lang.org"
internal = "https://packages.mycompany.com"

[dependencies]
serde   = "^2.1"                                  # default registry, semver
json    = { version = "1.0", features = ["serde"] }
internal-lib = { version = "3.0", registry = "internal" }

[dev-dependencies]
proptest = "1.0"
```

### 1.5 Dependency Sources

Dependencies can come from multiple sources. All are content-addressed in the lockfile.

```toml
[dependencies]
# ─────────────────────────────────────────────────────────────────────────────
# REGISTRY — curated, namespaced, semantic versioning
# ─────────────────────────────────────────────────────────────────────────────
serde = "^2.1"                                    # default registry
serde = "=2.1.5"                                  # exact version pin
serde = { version = "^2.1", registry = "internal" }
serde = { version = "^2.1", features = ["derive", "json"] }

# Semver ranges:
#   "^2.1"   = ">=2.1.0, <3.0.0"   (compatible updates)
#   "~2.1"   = ">=2.1.0, <2.2.0"   (patch updates only)
#   "=2.1.5" = exact pin
#   ">=2.1, <2.5" = explicit range

# ─────────────────────────────────────────────────────────────────────────────
# GIT — branches, tags, commits, monorepo paths
# ─────────────────────────────────────────────────────────────────────────────
mylib = { git = "https://github.com/foo/mylib" }                     # default branch
mylib = { git = "https://github.com/foo/mylib", branch = "develop" }
mylib = { git = "https://github.com/foo/mylib", tag = "v1.2.3" }
mylib = { git = "https://github.com/foo/mylib", rev = "a1b2c3d" }    # exact commit

# Monorepo: package is in a subdirectory
utils = { git = "https://github.com/foo/monorepo", path = "packages/utils" }
utils = { git = "https://github.com/foo/monorepo", tag = "v2.0", path = "libs/utils" }

# ─────────────────────────────────────────────────────────────────────────────
# URL — archives, vendored tarballs
# ─────────────────────────────────────────────────────────────────────────────
vendored = { url = "https://example.com/lib-1.0.tar.gz" }
vendored = { url = "https://example.com/lib-1.0.zip", sha256 = "a1b2c3..." }

# ─────────────────────────────────────────────────────────────────────────────
# CONTENT ADDRESS — immutable, hash is the identity
# ─────────────────────────────────────────────────────────────────────────────
pinned = { sha256 = "a1b2c3d4e5f6...", url = "https://..." }  # url is hint, hash is truth
pinned = { ipfs = "QmYwAPJzv5CZsnA..." }

# ─────────────────────────────────────────────────────────────────────────────
# PATH — local development, workspaces
# ─────────────────────────────────────────────────────────────────────────────
local-dev = { path = "../my-local-lib" }
```

### 1.6 The Lockfile

`Ferrum.lock` pins exact content. The source tells you where to find it; the hash tells you what it must be.

```toml
# Ferrum.lock — DO NOT EDIT. Generated by `ferrum build`.

[[package]]
name = "serde"
version = "2.1.5"
source = "registry+https://packages.ferrum-lang.org"
sha256 = "a1b2c3d4e5f6789..."

[[package]]
name = "mylib"
version = "0.0.0"
source = "git+https://github.com/foo/mylib?rev=deadbeef12345678"
sha256 = "b2c3d4e5f6g7890..."

[[package]]
name = "utils"
version = "2.0.0"
source = "git+https://github.com/foo/monorepo?tag=v2.0&path=libs/utils"
sha256 = "c3d4e5f6g7h8901..."

[[package]]
name = "vendored"
version = "1.0.0"
source = "url+https://example.com/lib-1.0.tar.gz"
sha256 = "d4e5f6g7h8i9012..."

[[package]]
name = "local-dev"
version = "0.1.0"
source = "path+../my-local-lib"
# No hash — path deps are always rebuilt from source
```

**Key properties:**

- **Content hash is king.** If the downloaded content doesn't match the lockfile hash, build fails.
- **Sources can disappear.** If you have the hash, you can find a mirror or use a content-addressed network.
- **Multiple versions allowed.** Two deps can depend on different versions of the same package. Both are built and linked separately.
- **Path deps have no hash.** They're always rebuilt from source — that's the point.

### 1.7 Overrides and Patches

```toml
# Replace a dependency globally (e.g., to test a fix)
[patch.registry.default]
serde = { git = "https://github.com/me/serde", branch = "my-fix" }

# Replace for a specific source
[patch."git+https://github.com/foo/mylib"]
mylib = { path = "../mylib-fork" }
```

### 1.8 Workspaces

Multiple packages in one repo:

```toml
# Ferrum.toml at repo root
[workspace]
members = [
    "crates/core",
    "crates/cli",
    "crates/server",
]

[workspace.dependencies]
# Shared dependency versions for all members
serde = "^2.1"
tokio = "^1.0"
```

Members inherit workspace dependencies:

```toml
# crates/core/Ferrum.toml
[package]
name = "myapp-core"
version = "0.1.0"

[dependencies]
serde = { workspace = true }
```

### 1.9 The Prelude

The following are imported in every module without explicit import:

```ferrum
// Types
Option[T]    Result[T, E]    String    Vec[T]    HashMap[K, V]
Box[T]       Arc[T]          Rc[T]

// Traits
Copy Clone Drop Display Debug
Eq PartialEq Ord PartialOrd
Iterator From Into
Send Sync

// Functions
print  println  panic  unreachable  todo  unimplemented

// Macros
assert  assert_eq  assert_ne  dbg  todo
```

## 2. The Standard Library

### 2.1 Core (`core`)

No allocation. No OS. Safe for `no_std` embedded targets.

```ferrum
core.ops.*          // operator traits
core.cmp.*          // Eq, Ord, etc.
core.iter.*         // Iterator and combinators
core.option         // Option[T]
core.result         // Result[T, E]
core.mem.*          // size_of, align_of, swap, replace
core.ptr.*          // raw pointer operations (unsafe)
core.slice.*        // slice operations
core.str.*          // &str operations
core.num.*          // numeric traits and conversions
core.intrinsics.*   // compiler intrinsics (unsafe)
```

### 2.2 Alloc (`alloc`)

Requires an allocator. No OS. Available in embedded with a custom allocator.

```ferrum
alloc.vec           // Vec[T]
alloc.string        // String
alloc.collections.* // HashMap, BTreeMap, BinaryHeap, etc.
alloc.boxed         // Box[T]
alloc.rc            // Rc[T]
alloc.arc           // Arc[T]
alloc.sync.*        // Mutex, RwLock, Condvar
```

### 2.3 Standard (`std`)

Full standard library. Requires OS.

```ferrum
std.io.*            // I/O traits and types  ! IO
std.fs.*            // Filesystem            ! IO
std.net.*           // Networking            ! Net
std.process.*       // Process management    ! IO + Unsafe
std.thread.*        // OS threads            ! Sync
std.time.*          // Clocks and durations
std.env.*           // Environment           ! IO
std.path.*          // Path manipulation
std.ffi.*           // C interop
std.panic           // Panic handling
```

### 2.4 Key Traits Reference

```ferrum
trait Clone {
    fn clone(self: &Self): Self
}

trait Copy: Clone {}   // marker — copy semantics

trait Drop {
    fn drop(self: &mut Self)
}

trait Default {
    fn default(): Self
}

trait Display {
    fn fmt(self: &Self, f: &mut Formatter): Result[()] ! IO
}

trait Debug: Display {}

trait Hash {
    fn hash[H: Hasher](self: &Self, state: &mut H)
}

trait Send {}   // marker: safe to send across threads
trait Sync {}   // marker: safe to share across threads (&T: Send implies T: Sync)

trait From[T] {
    fn from(value: T): Self
}

trait Into[T] {
    fn into(self: Self): T
}
// Blanket: impl[T, U: From[T]] Into[T] for U

trait Error: Display + Debug {
    fn source(self: &Self): Option[&dyn Error]
}
```
